# STREMIO M3U AUTO

Addon automático que lê listas M3U e cria categorias sozinho.

## COMO USAR

1. Substitua:
lista.m3u8

pela sua lista IPTV.

2. Envie para GitHub

3. Deploy no Render

4. Pronto

## LINK FINAL

https://SEUAPP.onrender.com/manifest.json

## FUNCIONALIDADES

- categorias automáticas
- logos automáticas
- canais ilimitados
- compatível Android TV
- compatível celular
- compatível PC
