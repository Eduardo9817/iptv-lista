# IPTV Stremio OneClick Deploy

Projeto pronto para deploy automático no Render.

## Deploy automático

1. Crie conta:
https://render.com

2. Envie os arquivos para o GitHub

3. No Render clique:
New + -> Blueprint

4. Escolha o repositório

5. Deploy automático

## Manifest

Depois do deploy:

https://SEUAPP.onrender.com/manifest.json

Cole no Stremio.

## Compatível com:
- Android TV
- TV Box
- Windows
- Linux
- Celular
